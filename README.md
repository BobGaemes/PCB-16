# PCB-16 Computer

just a custom 16 bit computer, totally not inspired by the 6502

- 8x registers
- 32KB RAM
- 32KB ROM
- 4 I/O Ports with interrupts
- Per port interrupt enable/disable
- Many ALU functions
- V/Z/N/C flags
- Global interrupt enable/disable
- 256 byte stack
- 47 instructions, 128 opcodes
- Multiple addressing modes (immediate, absolute, relative, absolute indexed, register)
- Reset + port interrupt vectors
